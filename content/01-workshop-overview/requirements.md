# Requirements

1. This lab will take you through a journey of using Istio on Kubernetes. A prerequisite of this lab is basic understanding of Kubernetes.

2. The workshop will be based on **Google Cloud Platform**. Labs accounts will be provided to run the workshop during the event. If you are running the workshop outside of the event, you have to create a Google Cloud Platform account.
